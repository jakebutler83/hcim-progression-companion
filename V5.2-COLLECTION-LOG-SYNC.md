# V5.2 Collection Log Sync

This version captures actual Collection Log page data from the in-game interface.

## First-time clue import

1. Start RuneLite in development mode with this plugin enabled.
2. Log into OSRS.
3. Open the in-game Collection Log.
4. Select **Treasure Trails**.
5. Open each page once:
   - Beginner Treasure Trails
   - Easy Treasure Trails
   - Medium Treasure Trails
   - Hard Treasure Trails
   - Elite Treasure Trails
   - Master Treasure Trails
   - Shared Treasure Trail Rewards
6. In the companion panel, confirm **Collection Log** shows captured pages/items.
7. Click **Sync Account Now**.
8. Refresh the website's Clue Scroll section.

The OSRS client only creates item widgets for the currently opened Collection Log page, so pages must be opened once per RuneLite session before a complete manual sync. Dink remains responsible for detailed future clue reward history.
