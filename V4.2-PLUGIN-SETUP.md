# HCIM Progression Companion v4.2

1. Replace your plugin project with these updated files.
2. In RuneLite plugin settings, set Website API URL to:
   https://YOUR-NETLIFY-SITE.netlify.app/.netlify/functions
3. Run `gradlew.bat clean build`, then `gradlew.bat run`.
4. Generate a code in Website Settings.
5. Enter the code in the RuneLite companion panel and click Connect.
6. Enable Share location.
